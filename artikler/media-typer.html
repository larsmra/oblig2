Media Typer

3. oktober 2018

Hva er mediatyper

Media typer forteller hvilke enheter som en nettside vises i[1]. Disse media typene er for eksempel mobiltelefoner, PC-skjermer, prosjektorer og utskrift på papir. Det finnes en del media typer. Her er en liste over noen av dem:

  • All: Dette er standard media typen, som gjelder hvis man ikke spesifiserer noe annet[1].

  • Screen: Denne spesifiserer at nettsiden vises på en skjerm[1].

  • Handheld: Brukes for håndholdte enheter, som mobiler og nettbrett[1].

  • Print: Brukes for utskrifter[1].

  • Projection: Brukes for prosjektorer[1].

Man kan bruke media typer for å gi nettsiden forskjellig utseende på ulike enheter[2]. Hvis man, for eksempel, vil skrive ut en nettside, vil man ikke ha med menyene, fordi de ikke brukes til noe på et papir. Da kan man spesifisere det i CSS slik: 

                                
        @media print {
            nav {
                display: none;
            }
        }

Hvorfor noen medietyper ikke brukes lenger

Problemet med å spesifisere mange ulike media typer i CSS, er at det fort blir mye CSS å styre med. I CSS3 ble det introdusert media queries, som kunne sjekke enhetens bredde, høyde og oppløsning[2]. Dette gjorde det mye enklere å designe nettsiden så den så bra ut på enheter med ulike størrelser. Media queries, sammen med et fleksibelt oppsett, er med på å lage et responsivt design[3]. Responsivt design har ført til at noen av media typene nevnt ovenfor, som handheld og projection, ikke er like mye brukt nå lenger.
